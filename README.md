# 📝 Note App API

A RESTful API for managing personal notes with secure user authentication using JWT.

## 🚀 Features

- User Registration
- User Login with JWT Authentication
- Protected Routes
- Create Note
- Get All Notes
- Get Note By ID
- Update Note
- Delete Note

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- dotenv
- CORS
